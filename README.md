## 🚀 Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. **Clone the repo**:
   ```sh
   git clone https://github.com/Jagadeeshftw/Next_Level_Portfolio.git
   ```
2. **Install dependencies**:
   ```sh
   cd Next_Level_Portfolio
   npm install
   ```

### Running the Project

To start the development server, run:

  ```sh
  npm run dev
  ```

To build the project for production, run:

  ```sh
  npm run build
  ```
